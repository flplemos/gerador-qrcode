# 📱 Gerador de QR Code Permanente

Um gerador de QR Code estático, rápido e altamente customizável construído com Vanilla JavaScript. Crie códigos para links ou textos com opções avançadas de estilo e baixe instantaneamente.

🔗 **[Acesse o projeto rodando aqui](https://flplemos.github.io/gerador-qrcode/)**

## ✨ Funcionalidades

* **Geração Instantânea:** Crie QR Codes para qualquer URL ou texto.
* **Customização Completa:**
  * Altere o tamanho da imagem gerada (de 128x128 até 1024x1024).
  * Escolha cores personalizadas para o código e para o fundo.
  * Ajuste o nível de correção de erro (L, M, Q, H).
* **Opções de Download:**
  * Baixar apenas o código puro (fundo transparente/sólido).
  * Baixar com borda (design em formato de "card" arredondado).
* **Modo Claro/Escuro:** Alternância de tema (`Light/Dark Mode`) com preferência salva automaticamente no navegador.
* **Interface Responsiva:** Funciona perfeitamente em desktops e dispositivos móveis.

## 🛠️ Tecnologias Utilizadas

* **HTML5** (Semântica e estrutura)
* **CSS3** (Variáveis, Flexbox, Grid e Media Queries)
* **JavaScript (Vanilla)** (Lógica de geração, manipulação de DOM e Canvas)
* **[QRCode.js](https://davidshimjs.github.io/qrcodejs/)** (Biblioteca para renderização do código)
